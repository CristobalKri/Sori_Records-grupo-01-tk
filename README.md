Objetivo: Crear una app que permita ofrecer productos, gestionar su disponibilidad y registrar compra o solicitudes.

Estructura:

Carrito de compras (agregar, eliminar, confirmar pedido)
CRUD de productos (discos)
Gestión inventario adminHistorial de compras o solicitudes por parte del cliente.
Visualización de detalles de productos (discos de música)
Validaciones para agregar o actualizar inventario, catálogo y usuarios.


Roles:

Cliente: Navegar y comprarAdmin: Gestión de catálogos, inventario y pedidos
